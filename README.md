# Leaf World
## This is a website inspired by the theme "fall". It's a mini world where you can farm leaves, collect daily bonus and also buy stuffs from the shop. 

## Features
* Farm leaves
* Claim daily bonus
* Buy stuffs from the FALL shop
* View bought items

## Technologies used
* JavaScript 
* HTML and CSS
